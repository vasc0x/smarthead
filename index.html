Welcome to our team project.

Here you'll find all the information relating to both the software and hardware development of our Electrical Engineering Senior Project at Florida International University, FIU, in beautiful Miami, Florida, United States. This is a work in project so, please bookmark us and check often!

Open software

We are using GitHub to store all the code relevant to the project. Both the C code for the AVR microcontroller and the PHP/HTML/CSS for the data collection website.
